
# Glow.tf | Projeto Web

**Glow.tf** é o protótipo de um site de vendas de skin de Team Fortress 2. Projetado como um trabalho em grupo da disciplina de Programação Web, do curso de Ciência da Computação da PUCPR.

## Conteúdo

- **Modo Administrador**
  
- **Lista de produtos do site**

- **Adicionar produto**

- **Editar produto**

- **Deletar produto**

- **Home**

- **Carrinho**

- **Lista de Desejos**

- **Login**

- **Cadastro**

- **Perfil do Usuário**

- **Pagamento**

- **Página do Produto**

- **Identificação do Usuário**

### Ferramentas

- **HTML5**
- **CSS**
- **PHP**
- **JavaScript**
- **MySQL**
- **Xampp**
